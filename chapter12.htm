12.3.     Cross-plot tabs

There are eight primary tabs in the cross-plot tab that help you display the cross-plot: Well, Configuration, Zone, Formula, Regression line, Layers, Overlay line, Print Setting.

12.3.1.         Well tab

On this tab, you can drag and drop wells from Project Browser to the cross-plot area. The wells will be displayed on this tab. You place curves to be displayed by specifying the X-Axis, Y-Axis, Color Z1, Point size Z2, and Label Z3.

Top of the tab, you can filter variables by the Family Group, Family and Curve name.

The tab displays the POR values (X-Axis) versus PERM values (Y-Axis). Optional dimensions include:

·      Color Z1 axis, related to the Z1 value

·      Point size Z2, related to the Z2 value

·      Label Z3, related to the Z3 value

After variables are selected, you have to click on the Update button at the bottom of the tab to display data over cross-plot.

Figure 12‑5 Well tab in cross-plot

12.3.2.         Configuration tab

This tab allows you to modify cross-plot display after settings curves selection in the Well tab.

·      Configuration tab allows you to set up the scale of the X-axis and Y-axis, the left scale and right scale are automatically set from curve family and you can manually change by typing preferred value. You can also define by typing a positive number of grid lines for major and minor.

·      Color Mode is used to view data color. You can choose to view color by zone or well.

·      I2G also provides a lot of information, such as the number of points, MSE, Correlation…

Figure 12‑6 Configuration tab in cross-plot

12.3.3.         Zone tab

12.3.4.         Formula tab

This tab allows assists users to add lines on any cross-plot to enter an equation. Here, you will create an equation and display it in a cross-plot.

You enter an equation into a text box to create a line. In order for the equation to calculated, it must be written with the following syntax below:

It is possible to edit/save equations at any time during the operation. To edit an equation, re-open the User define line form and then you can modify or create a new one.

12.3.5.         Regression line tab

Regressions are equation lines displayed through the data points. Regressions can be added to fit the displayed data or inserted graphically over the plot.

Generate a regression by active layers on the tab. The layer can be zones or wells depend on user setting
