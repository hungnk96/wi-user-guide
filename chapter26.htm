Module 26:  Python

Python is a programming language that can be used as the script in i2G. It supports you to create your own programs or functions, batch data management, plotting or even new modules.

26.1.     Log into Python application

Step 1.         From Menu Bar, click on Home tab and choose Python. The Python tab appears next to the i2G tab on the right.

Figure 26‑1   Open Python tab

You can also access directly the Python application via https://python.i2g.cloud/

Step 2.         At the Login dialog, enter your Username and Password

Figure 26‑2 Log into Python application

Step 3.         Click Login.

Note: If you’re having trouble logging in, please contact your administrator or contact us via https://www.i2g.cloud/contact-us/

26.2.     The components of the Python application

Figure 26‑3 The components of the Python application

26.2.1.         Tool Bar

At the top of the Python window you will see the Tool Bar, where you can access a range of actions essential to your Python Project.

Table 26‑1 Tool Bar in the Python window

Name of button

Description

Project

There are 4 options: Add new project, Open an existing project, Close project and Delete project

New

Add new folder or new script file to the treeview

Action

Run or save the current selected script

Tools

Login by token or by account. Users can also activate dark mode for the window here

Help

View API document, i2G family base or Python version information

Logout

Log out of Python application

 

26.2.2.         Python Script list

The left data-tree lists all script files of the current Python project. Focus on a folder name and expand all the nodes in this tree view control. You can see a hierarchical list of files

Figure 26‑4 Python Script list

The menu of the Python Script list contains the following functions:

·      Delete: delete a script file/folder

·      Rename: rename a script file/folder

·      Search: Search for text in the list. Search results will be highlighted in blue.

26.2.3.         Code Editor

The Code Editor displays python data in a code editor. The editor is capable of formatting and inspecting python.

Figure 26‑5 Code Editor in Python app

The menu of the Code Editor contains the following functions:

·      Run Code: run code. You can also run code by choosing this option from the pop-up menu of Action button on Tool Bar

·      Save Code: save code. You can also run code by choosing this option from the pop-up menu of Action button on Tool Bar

·      Auto Save: automatically saves the current changes

·      Save Reminder: Remind users to save a script if it is not saved before closing

Tip: to save time, drag-and-drop data (project/well/dataset/curve) directly from Project list into the Code Editor to get its information instead of manually typing it.

26.2.4.         Console

The Console manages outputs for Python application.

Figure 26‑6 Console window in Python app

The menu of the Code Editor contains the following functions:

·      Terminate: terminate a process

·      Clear: clear screen in the Console

26.2.5.         Project list

The right data-tree lists all projects of the active user. Focus on a project name and expand all the nodes in this tree view control. You can see a hierarchical list of data

Figure 26‑7 Project list in Python app

The menu of the Project list contains the following functions:

·      Reload: reload to get the latest data of the projects

·      Sort by name: arrange a list in alphabetical order

·      Search: Search for text in the list

Tip: drag-and-drop any data (project/well/dataset/curve) directly from this tree into the Code Editor to get its information.
