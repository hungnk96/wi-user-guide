Learning objectives

After completing this module, you will know how to:

·      Create a histogram, appearance of the histogram interface

·      Manipulate the plot from tabs in the interface

·      Work with stack and none stack function in the histogram

13.1.     Create a new histogram

There are three ways to create a Histogram (multi-well)

·      Select a curve in log track, right-click on them and then select Create Histogram option

Figure 13‑1 Create histogram from log-plot

 

·      From the Plot tab on the menu bar, go to the Histogram section. You can select choose Blank Histogram to build your own

Figure 13‑2 Create histogram from Tool Bar

·      From the Project Browser, go to the Histogram section. You can right-click and choose Blank Histogram option

Figure 13‑3 Create histogram from Data tree

13.3.     Histogram tabs

There are seven primary tabs in the histogram tab that help you display the histogram: Well, Configuration, Zone, Assistive lines, Statistic, Layers, and Print Setting.

13.3.1.         Well tab

·      On this tab, you can drag and drop wells from Project Browser to the histogram area. The wells will be displayed on this tab. You place curves to be displayed by specifying the X-Axis.

·      Top of the tab, you can filter variables by the group family, family and curve name.

·      After variables are selected, you have to click on the Update button at the bottom of the tab to display data over the histogram

Figure 13‑5 Well tab in a histogram

13.3.2.         Configuration tab

This tab allows you to modify histogram display after settings curves selection in the Well tab.

·      Configuration tab allows you to set up the scale of the X-axis, left scale and right scale is automatically set from curve family and you can manually change by typing preferred value. You can also define by typing a positive number of grid lines for major and minor.

·      Color Mode is used to view data color. You can choose to view color by zone or well.

 

 Tips: You can also change the data display by using Stack Mode options. The Stack Mode options include No Stack, Well, Zone, and All (Figure 2.6).

Figure 13‑6 Configuration tab in a histogram

13.3.3.         Zone tab

By default, Histogram will display the ZonationAll. You can also define another zone set by click on the zone set dropbox to choose one

Figure 13‑7 Zone tab in a histogram

13.3.4.         Assistive Line

This tab allows you to view data as Gaussian and Cumulative by clicking on each layer. A layer can be zone or well depend on display setting

Figure 13‑8 Assistive line tab in a histogram

13.3.5.         Statistic tab

I2G supports a lot of statistic information, for example, P10, P50, P90, Min, Max, Average…

Figure 13‑9 Statistic tab in a histogram

13.3.6.         Layers tab

This tab allows you to filter data display on a histogram. Layers can be zones or wells. You click on each layer to turn them on/off

Figure 13‑10 Layer tab in a histogram
