Module 5:       Data Export

         There are 2 exportation types:

o   Export from inventory

o   Export from Project

Note: If you can't export data, the owner may have disabled the option to export

5.1.1.   Export from inventory

            In this type, the user will select the data to be exported to their computer from Inventory:

Step 1.         In the Data tab from the Menu Bar, click Export then select Export from Inventory. The Export from Inventory window appears

Figure 5‑1 Open Export from the Inventory window

Step 2.         Select the desired wells/datasets/curves in the Inventory area then click (  ) to move them to the Export Queue window

Figure 5‑2 Move data from Inventory to Export Queue

Step 3.         At the top right corner, set the file type (.Csv, Las2, Las3 or Dlis) user want to export

Figure 5‑3 Select export file type

Step 4.         Click (  ) to export all the selected wells from the Export Queue to in temporary memory.

Figure 5‑4 Export data from Export Queue to the temporary memory

          Tips: users can remove data from the Export Queue area by selecting them and select (  ) or () to remove all.

Step 5.         To download files to your computer, click on exported filenames or () to download all

 

 

5.1.2.   Export from project

         In this type, the user will export data directly from the project main interface.

Step 1.         In the Data tab from the Menu Bar, click Export then select Export from Project. The Export from the Project window appears

Figure 5‑5 Open Export from Project window

Step 2.         Select the desired data in the Inventory area then click (  ) to move them to the Export Queue window

5.1.3.   Export Zone

Step 1.         From Menu Bar, click on Data tab and choose Zone Set Manager, the Zone Set Manager window will be displayed

Figure 5‑9 Open Zone Set Manager window

Step 2.         At the top left corner, click Export zone set button (  ). Select wells to export dialog pop up

Figure 5‑10 Select wells to export zone sets

Step 3.         Select wells you want to export then click OK

Step 4.         Select the Zone Set name and the Unit to be exported in the pop-up windows then click OK.

Figure 5‑11 Select the Zone Set name to export

 

 

5.1.4.   Export Marker

Step 1.         From Menu Bar, click on Data tab and choose Marker Set Manager, the Marker Set Manager window will be displayed

Figure 5‑12 Open Marker Set Manager window

Step 2.         Select Export Marker set button. Select wells to export dialog pop up

Figure 5‑13 Select wells to export marker sets

Step 3.         Select wells you want to export then click OK.

Step 4.         Select the Marker Set name and the Unit to be exported in the pop-up Select Marker set to export window then OK

Figure 5‑14 Select the Marker Set name to export

 

 

5.1.5.   Export Asset

Step 1.         From Menu Bar, click on the Data tab and choose Asset Manager, the Asset Manager window will be displayed

Figure 5‑15 Open Asset Manager window

Step 2.         Right-click on an Asset you want to export, select Export from the context menu

Figure 5‑16 Export Assets

Tip: To download multiple assets, press Ctrl while clicking other assets

Step 3.         Choose where you want to store the files on your desktop, then enter a name and click Save. The exported files will be in Json (.JS) format.
