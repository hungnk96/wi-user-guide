Module 25:  Base Map

The Base Map is a visual mapping tool that represents the geographical position of the wells.

25.1.     Log into Base Map

Step 1.         From Menu Bar, click on Home tab and choose Base Map. The Base Map tab appears next to the i2G tab on the right.

Figure 25‑1   Open Base Map tab

You can also access directly the base map via https://base-map.i2g.cloud/

Step 2.         At the Login dialog, enter your Username and Password

Figure 25‑2 Log into Base Map

Step 3.         Click Login.

Note: If you’re having trouble logging in, please contact your administrator or contact us via https://www.i2g.cloud/contact-us/

25.2.     The components of the Base Map

Figure 25‑3 Base map interface

25.2.1.         Menu window

By default, the Menu window is visible and appears on the left of the map.

Figure 25‑4 Setting area in Base Map

On the top right of the Menu window, click () to log out of Base Map. Click () to collapse the window.

Table 25‑1  Buttons in Menu window of Base Map

Icon

Name of Icon

Description

Dashboard

Show Dashboard on the map

Use

Choose a project/well from the Project tab and click () to put this one to the Wells tab. This button is only activated provided that you are in the Project tab and a project/well is selected

Delete Wells

Select desired wells and click () to delete these selected wells.

Clear

Remove all wells in the Wells tab. This button is only activated provided that you are in the Wells tab.

Reset

Reset all settings to default

 

There are 4 tabs in the Menu window:

25.2.1.1.    Project tab

All projects and wells data of the current user will be listed in this tree view control. You cannot modify these data here.

25.2.1.2.    Wells tab

All of the wells that were selected from Project tab will be listed here. Focus on a well name and the   shows where the well is.

25.2.1.3.    Appearance tab

Appearance tab allows you to specify theme, dark mode for the Menu window. You can also hide/unhide the Control Panel, the Pop-up wells name and the Point Location of your mouse.

25.2.1.4.    Configs tab

Configs tab allows you to display zone set, marker set, contour or boundaries on the map.

25.2.2.         Control Panel

Base Map contains elements to allow user interaction with the map. These elements are known as controls. By default, this Control Panel is visible and appears in the top left corner of the map.

Table 25‑2  Controls in base map

Icon

Name of Icon

Description

Full-screen

Expand to full screen, hiding the Menu window and Control panel.

Street view

Interactive panoramas from positions along many streets in the world. Just drag Pegman to a place on the map to view it.

Zoom In

Increase the zoom level of the map

Zoom Out

Decease the zoom level of the map

 

Tips:

-       To look around, click and drag your mouse

-       To zoom in or out, scroll with your mouse or two-finger zoom with a touchpad. You can also use (+) and (-) on the Control panel

-       To position your Street View north, just click the compass.

25.2.3.         Map

A Map is an interface that displays all objects on map tiles and world coordinates.

Hold down the Ctrl key while you are scrolling to zoom the map in and out. As you zoom in, additional data and insights become available. You can also use pinch, rotate, and tilt to explore the map in more details.

Tip: You can change the map theme to change the way your map looks in Appearance tab

25.1.     Interact with your map

25.1.1.         Display Zone set and Marker set on Base Map

You can display zone’s coordinates or marker’s coordinates instead of wellhead’s coordinates.

Step 1.         From Menu window, choose Configs tab

Step 2.         Turn on Display Marker Set On/Off switch (). The switch will be in blue

Figure 25‑7 Display a marker’s coordinates on the map

Step 3.         Select a marker you want to display on the map

Figure 25‑8 Well location base maps at a horizon

25.1.2.         Display contour

Step 1.         From Menu window, choose Configuration tab

Step 2.         Turn on Show Contour On/Off switch (). The switch will be in blue

Step 3.         Select a curve for contour rendering

Figure 25‑9 Display contour on the Base map

25.2.     Boundary shapefile

I2G allows users to upload a shapefile (*.zip) to understand your data better by visualizing boundaries on map.

Step 1.         From Menu window, choose Configuration tab

Step 2.         Click Upload boundary file () and select a file from the window browser. Then click Open.

Figure 25‑10 Browse and navigate to the shapefile *.zip location

Figure 25‑11 Display boundary shapefile on the base map

25.3.     Save and apply a configuration file to Base Map

I2G allows users to save all settings of Base map for the later use

25.3.1.         Save configuration file of Base Map

Step 1.         From Menu window, choose Appearance tab

Step 2.         Click Save Configuration file and choose a location to store your file.

Step 3.         In the File name box, enter a new name for the file.

Step 4.         Click Save

25.3.2.         Apply your configuration file to Base Map

Step 1.         From Menu window, choose Appearance tab

Step 2.         Click Open configuration file and select a file from the window browser.

Step 3.         Click Open
