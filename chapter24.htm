Module 24:  Depofacies Predictor

This module is an AI built to predict depositional facies. A new hybrid approach which combines machine learning and an expert system generates data-driven, reproducible and consistent results for facies prediction across an entire field. Unlike most machine learning models, the prediction is made considering log data in the context of depositional units. First, the Gamma Ray curve is divided into different depositional units by tracking Gamma Ray value changes. The machine then automatically assigns to each depositional unit a Gamma Ray shape, lithofacies, boundary type and its stacking pattern with adjacent units. These parameters are integrated with a machine learning model built on core data from the studied basin plus relevant biostratigraphic data and then passed through an expert system containing a set of fuzzy rules to yield final probability curves for each depositional facies. After making the initial facies assignments, the facies above and below each unit, along with the style of each unit boundary, and the range of associated facies, are determined by looping back through the expert system to produce a coherent stratigraphic succession. Outputs include predictions of the three most likely depositional facies for each unit and their respective probabilities.

There are two tasks in the workflow: “Unit Breakdown” and “Expert Rules”.

24.1.     Unit Breakdown

The purpose of this task is to divide the studied section into different depositional units. The output will be “Boundary Flag” which marks the boundary between adjacent depositional units. The minimum thickness of a depositional unit is 1 mTVD and so TVD curve is a mandatory input for this task.

Unit Breakdown task follows a Task frame including General, Input Configuration, Output Configuration and Input Selection tab

Step 1.         From the Menu Bar, select Machine Learning tab > select Unit Breakdown

 

Figure 24‑1 Open Unit Breakdown task from Menu Bar

Step 2.         In the Input Configure selection window, choose these families: Gamma Ray and True Vertical Depth

Step 3.         In the Input Selection window, drag the well into the Input area.

24.2.     Expert Rules

The purpose of this task is to take all available information and proceed them through an expert system to come up with the results (35 probability curves) in forms of fuzzy logic.

Expert Rules task follows a Task frame including General, Input Configuration, Output Configuration and Input Selection tab

Step 1.         From the Menu Bar, select Machine Learning tab > select Expert Rules

Figure 24‑3 Open Expert Rules task from Menu Bar

Step 2.         In the Input Configure selection window, choose these families:

·      Mandatory input: Gamma Ray, Unit Boundary, True Vertical Depth & Mud Contents

·      Optional input: Biostratigraphic Interpretation, Reliability of Biostratigraphic Interpretation, Special Lithology, Core Depofacies, Lateral Proximity and ML-derived Facies

Step 3.         In the select Input Selection window, drag the wells into the Input area.

Step 4.         Select the proper input variables for the well

Figure 24‑4 Expert Rules task configuration

Step 5: Under Task, Select button    for the computation. Button    will appear if the user checked the Output curves and log-plot checkbox in General Section. A log-plot layout will open by clicking the well name.

10

GR_SHAPE_CODE

-       Coarsening upward (CU): Decrease GR value from bottom to top ("upward")

-       Finning upward (FU): Increase GR value from bottom to top ("upward")

-       Uniform (UN): Constant GR value - not much change in grain size

-       Serrated (SR): significant value change(s) in < 1 m of vertical succession. Can be CU, FU or UN

STACKING_PATTERN

Defines the grain size trend of adjacent units. In other words, stacking pattern concerns about how adjacent units within a parasequence are stacked together.

The values can be Aggradation (AG), Fining upward (FU) and Coarsening upward (CU).
