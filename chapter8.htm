Module 8:       Image Manager

Learning Objectives

After completing this module, you will know how to:

·      Manage images in a project with the Image Manager form

·      Manually create an Image Set for a well

·      Display Image Set in log-plot

8.1.        Organize Image Set

To open Image Manager: From Menu Bar, click on the Data tab and choose Image Manager, the Image Manager window will be displayed.

Figure 8‑1 Open Image Set Manager window

·      An Image Set is a collection of one or many images.

·      Each image is defined by its information such as name, top depth, and bottom depth.

·      Whenever the information of the image has been changed, the image line will be turned blue, click Save (  ) icon to formally save the new data, click Reload () icon to cancel all the changes.

·      Users can delete multi-images by holding Shift/Ctrl key > click Delete ( ) icon to delete images, all the images you specifically delete will be greyed > click Save (  ) icon to formally save the new data, click Reload () icon to cancel all the changes

8.2.        Manually create an Image Set for a Well

Besides the way to import images by using Import Image Wizard dialog, the user can manually create an image set for a well, following procedure:

Step 1.         Focus on well ESS that you want to create a new image set > click Add Image Set () icon on the top navigation bar, New Image Set dialog pops up

Figure 8‑2 Add new Image Set for a Well

Step 2.         Type the name “UV” for the new Image Set > click Ok. You can see the “UV” image set is added below well ESS

Step 3.         Focus on “UV” set > click Add Image () icon on the top right navigation bar. A new image will be listed in the table

Figure 8‑3 Add Image

Step 4.         Specify name, top depth & bottom depth of the new image

Step 5.         Click Upload, then Choose Image dialog pops up.

Figure 8‑4 Open “Choose Image” dialog

Step 6.         Click Choose Image and navigate to your image folder. Choose an image you want to load and click Open. The selected image will be shown in the dialog

Step 7.         Click Upload to import the image into i2G.

Figure 8‑5 Preview an Image before uploading

Step 8.         The new image line will turn green, click Save () icon to formally save the new data
