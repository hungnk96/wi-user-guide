Module 10:  Tags

10.1.     Use tags to organize objects on Data Tree

You can tag any objects on data tree such as well, dataset, curve, plot, zone, marker, workflow… to make them easier to find.

How to assign tags to objects:

Step 1.         On data tree, right-click on objects you want to assign tags > click Assign Tags. The Tags dialog pops up.

Figure 10‑1 Open Tags dialog

Step 2.         Click in the Tags field, then enter a new tag, or choose one from the existing tags list > Click Ok.

Figure 10‑2 Enter a new tag or choose one from the existing tags list

Tips: You can assign multiple tags for multiple objects at once.

10.2.     Find objects you tagged

Step 1.         On the Project Browser tab, click on Tags () icon to open Tags window

Figure 10‑3 Open Tags navigation bar

Step 2.         Click on the name of any tag in the left navigation bar. Data tree will then only show the objects that are tagged with that tag.

Figure 10‑4 Tags on Data tree

10.3.     Remove tags

10.3.1.         Remove tags from an object

Open Tags dialog by right-clicking on an object > choose Assign Tags. Then click Remove () icon on the left side of each tag to remove a tag from an object.

Figure 10‑5 Remove Tags from an object

10.3.2.         Remove tags from the project

In the Tags window, click Remove () icon on the left side of each tag to remove a tag from your project.

Figure 10‑6 Remove Tags from project
