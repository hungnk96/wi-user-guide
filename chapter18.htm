18.3.     Baseline Shift

Learning objectives

The baseline shift tool measures significant changes to a system against previous references.

The equation used to shift the variable curve according to the graphically defined points is: LOG_RESULT = LOG + (BASELINE_ORIGIN – BASELINE)

After completing this module, you will know how to:

·      Manually create a baseline-shifting window

·      Baseline-shift a variable and a dataset

·      Save and Load a baseline-shifted curve/dataset

·      Save and Load a baseline-shift setting parameter table

18.3.1.         Creating a baseline shifting window

        This procedure shows you how to create a baseline-shifting window

Step 1.         Choose a Curve from data tree.

Step 2.         Drag-and-drop the curve in an empty log track.

Step 3.         Select the to-be-shifted curve > right-click on it and select Baseline Shift. The Baseline Shift window opens next to the current log-plot

Figure 18‑12 Creating a baseline shift window

18.3.2.         Baseline shifting a log

Baseline shift window has 2 tracks:

·      The first track (Origin) displays the original log. In this track, the log cannot be modified.

·      The second track (shifted curve) displays the baseline-shifted log. It is only in this track that the curve can be baseline-shifted

After you create the Baseline-shifting window, you will be able to manually drop a curve for reference.

§  To pick the first shift point, Ctrl + Left-click on the log at the Origin track. A red shift point on a blue baseline will be created along the Origin track. Users can also shape the baseline by clicking on the Origin track to add more shift points.

§  Move the baseline left or right to shift the curve. The baseline curve (*_BLS) will be changed and updated automatically

§  At the right side, the baseline shift table gives information about depth position of every single point. It includes value and depth position of shifted point. You can manually enter a depth value, the shift point on log-plot will be updated automatically

Figure 18‑13 baseline shifting a log

Note (No*): If distant between two points is smaller than the step of the input dataset. The point will be removed.

Tips:

§  There are two ways to remove a shift point. Click on it from baseline shift table, the selected shift point will be highlighted in the shift points table. Choose the delete icon (). The 2nd way, from log-plot you click on shift point, right - click and choose Delete.

§  Ctrl + scroll mouse to change interval scale. It helps you focus on a small interval in detail.

§  Because of working in the cloud, so it is possible to zoom in or zoom out chrome at any time during the operation.

18.3.3.         Saving the baseline shift

     After you are satisfied with a baseline-shift, you can save the curve and save the applied base-shifted configuration.

18.3.3.1.    Save a baseline-shifted curve

Step 1.         Right-Click on the shifted curve > choose Save Curve (1)

Step 2.         Choose the dataset to save the curve to in the Open Dataset pop-ups or create a new dataset to put the curve in > OK (2)

Step 3.         Choose the name for the saving curve, by default the log curve will automatically take the name of the curve with the suffix _BLS. Choose OK (3) to save the log and return to the current window

Figure 18‑14 Save baseline-shifted curve

18.3.3.2.    Save baseline-shifted configuration

Step 1.         Click the Save Icon () at the top right corner. A pop-up window will appear for you to choose the config name to be saved. By default, the config name will be automatically created

Figure 18‑15 Save baseline-shifted configuration

Step 2.         Choose OK to save the configuration and return to the current window

Note: all of the Baseline Shift configuration will be stored at Asset Manager

18.3.4.         Applying a baseline shifting

      This function allows you to load and apply a saved base-shift configuration to a variable or a dataset.

18.3.4.1.    From Baseline Shift window

Step 1.         In the Project Browser, choose the curve to shift and drag & drop it to the log-plot window

Step 2.         Right-click on the log, choose Baseline Shift. The Baseline Shift window will appear

Step 3.         Choose the load icon (). The open parameter set pop-up window will appear.

Step 4.         Select the configuration and choose OK

Figure 18‑16 Apply a saved baseline-shifted configuration to a curve from baseline shift window

18.3.4.2.    From the Project Browser

Step 1.         In the Project Browser, right-click on a dataset to shift. Choose Apply Baseline Shift table

Step 2.         Select the saved configuration to load in the saved Config Pop-up.

Step 3.         The Baseline Shift Config will appear in the pop-up Baseline Shift Config window. You can reselect the interpolation method, save mode and output suffix if needed. Confirm by choosing OK.

Step 4.         Select the name for the dataset to be saved to (for curves)/ the newly created dataset

Figure 18‑17 Apply a saved baseline-shifted configuration to a dataset from Project Browser

18.12. TVD Computation

The TVD computation tool generates a TVD from the measured depth, the azimuth and the inclination of a deviated well

·      MD: Measured Depth

·      TVD: True Vertical Depth

·      TVDSS: True Vertical Depth Subsea

·      Elevator/KB/DF: Kelly Bushing/Drilling Rig Floor

·      X Reference: Offset in the X direction from the well position, measured from sea level

·      Y Reference: Offset in the Y direction from the well position, measured from sea level

Figure 18‑52 Reference depths

2.1      

18.12.1.     Configure TVD Computation Task

Step 1.         From the Data Processing tab, select TVD Computation.

Figure 18‑53 Open TVD computation window

Step 2.         In the Input Configuration selection window, choose these families: Hole Deviation and Hole Azimuth

Figure 18‑54 Input configuration for TVD computation task

Step 3.         In the Input Selection window, drag wells into the Input area > select the proper input variables

Step 4.         On the Parameter tab, select the Minimum Curvature equation to be used.

 Tip: User can choose different formulas including Average Angle, Balanced Tangential, Radius Curvature, Minimum Curvature

Step 5.         Set the TVD Mode, Elevator, X reference, and Y reference parameters. Then Click Run

Figure 18‑55 Define parameters for TVD computation task

18.12.2.     Display TVD/TVDSS on the log-plot

18.12.2.1.Display TVD/TVDSS as log-plot reference

Step 1.         Focus on the log-plot area. Log-plot’s properties will be shown in the Properties Editor window

Step 2.         In the Depth Reference list box, select TVD/TVDSS. The whole log-plot will be reloaded with the TVD/TVDSS as reference

Figure 18‑56 Display TVD/TVDSS as log-plot reference

18.12.2.2.Display TVD/TVDSS in Depth track

Step 1.         Focus on Depth Track. Depth track’s properties will be shown in the Properties Editor window

Step 2.         In the Depth Reference list box, select TVD/TVDSS

Figure 18‑57 Display TVD/TVDSS in depth track
