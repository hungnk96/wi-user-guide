Module 11:  Log-Plot

      Log-Plot is a useful tool that allows you to display every petrophysical type of data, including log, zone, marker, text data, array data, core image, and core measurements of the Project Browser.

11.1.2.         Reference Track

Reference Track is always the first track of the log-plot window to quickly navigate along the borehole.

Click on this Reference Track to display log plots properties in the Properties Editor window

11.1.2.1.    Reference Curve of the log-plot

From Data tree, drag-and-drop a Curve you want to display as Reference Curve into the Reference Track. Reference Curve is available for all curves (except Text Curve).

Figure 11‑2 Reference Curve of the Log-plot

You can display the Reference Curve by the family’s curve’s scale.

11.1.2.2.    Navigation Bar

Figure 11‑3 Navigation Bar of the Log-plot

By moving the Navigation Bar in Reference Track, the user can quickly navigate to any part of the log display in the main log plot window. The user can change the depth interval being displayed by drag the top and bottom of the Navigation Bar up or down to a new position.

Tip: You can also change the Log-plot Scale by holding Ctrl while scrolling.

11.1.3.         Horizontal Scrollbar

Horizontal Scrollbar is located at the bottom of the log plot window, moves the display horizontally in increments of one track. Normally, it is better to simply expand the whole window if some of the tracks are not displayed

11.1.4.         View Controller

View Controller button () is located in the bottom right corner of the log plot window. Click on it to view all the Wells that are presenting in the current log-plot.

Users can hide Wells from the Log-plot by clicking on Well-name in the View Controller

Figure 11‑4 View Controller

Click Delete () icon next to the Well-name to delete all data of that well

11.1.5.         Blank Space

Click on this Blank Space to display Log-plot’s properties in the Properties Editor window.

11.1.6.         View Space

11.1.6.1.    Header of Log-plot

Each Log-plot has tree headers: Track, Variable, and Shading.

Figure 11‑5 Header of Log-plot

·      Track Header: display Track name and Well name.

·      Curve Header: display Curve name, unit, and scale of the curve.

·      Shading Header: display Shading name, type, and color of the shading

The Header can be resized by clicking on the bottom border of the header area and dragging it up or down to a new size.

11.1.6.2.    Tracks

A Track is a vertical section of a log presentation over which one particular set of data is displayed. I2G supports 5 track types: Depth track, Log track, Image track, Zonation track, and Object tracking.

11.1.6.2.1.                 Depth Track

Depth Track is displayed according to the available and selected datum such as MD, TVD, TVDSS. By default, the MD reference depth (in general the measured depth) is displayed in Depth Track.

Figure 11‑6 Depth Track

The depth value is displayed in the center of Depth Track.

Depth ‘ticks’ is also displayed on left and right of depth track as default

Users can change the properties (Depth type, color, unit, …) of Depth Track in the Properties Editor window

Figure 11‑7 Depth Track properties

Note: you can only change the setting of the Ticks from the Log-plot properties in the Properties Editor window

Figure 11‑8 Setting of the Ticks of Depth Track

11.1.6.2.2.                 Log Track

Log Track is a track where curves are displayed. When you add/drag a curve into a log-plot, the curve is automatically displayed in a Log Track.

Figure 11‑9 Log Tracks

The value ticks of the log track depend on the curve type that you are focusing on.

Figure 11‑10 Log Track types

11.1.6.2.3.                 Image Track

Image Track is a track where a set of images along the wellbore are displayed. Refer to this article for more information.

Figure 11‑11 Image Track

11.1.6.2.4.                 Zonation Track

The Zonation Track displays correlation line(s) between wells. If several wells are displayed in the Log-plot, you can insert a zonation track for each well. Refer to this article for more information

Figure 11‑12 Zonation Track

11.1.6.2.5.                 Object Track

Histograms and Cross-plots can be added to the Object Track. Refer to this article for more information.

Figure 11‑13 Object Track

11.1.6.3.    Curves

I2G supports 4 curve types: Log, Core, Text Curve and Array data. To display data, select and drag one or multiple curves from the Project Browser to a Log-plot window. The curves for each family display according to the parameters set in the family database.

Figure 11‑14 Variable types in i2G

 

Baseline shift: uses graphically define points as new references. The variable curve is shifted accordingly. Refer to … for more information.

 

Log Plot consists of Menu Bar, Navigation Bar, and View Space as shown below (Figure 4.1.1).

      I2G has 2 places to display the properties of an object:

q Properties window: Located under Project Browser window, Properties window structure may vary depending on the object(s) being selected at that moment

q Properties dialog: double-click or right-click on an object and select object properties option to open it. In this dialog, every option and setting will be displayed including advanced, complicated ones. With this, every object in the log view is fully customizable

11.2.1.4.    Edit Track properties

      I2G has 2 places to display properties of the track:

·      Properties window: focus on a track (the track will be highlighted in yellow), now you will see its properties in the Properties Editor window

Figure 11‑17 Track Properties in Properties Editor window

·      Properties dialog: double-click or right-click on a track and select curve properties option to open it.

Figure 11‑18 Open Track Properties dialog

§  Show title: allows the user to tick on/off to display the title of the log track.

§  Title: Number is displayed as default. The title of the log track allows the user to change the name by manually typing text.

§  Top justification: allows the user to change the alignment of title of depth track by selecting 3 options: left, right and center

§  Show value grid: allows the user to tick on/off to display the vertical grid line of log track. Tick on to display the vertical grid line and tick off to un-display the vertical grid line. It is not necessary to define the linear/logarithmic track, I2G will identify them automatically

§  Major ticks: to set up the major gridlines on the log track, allows the user to type number. The default value is 1.

§  Minor ticks count: allows the user to set up the minor gridlines on the log track, the default value is 5.

§  Show the depth grid: allows the user to tick on/off. Tick on the display of the horizontal grid lines of the log track. The horizontal grid line is identical to the major ticks of depth value. Tick off to un-display the horizontal depth grid lines.

§  Width: allows the user to change the width of log track by manually typing number

§  Width Unit: sets the unit of track width as inch or cm

§  Track color: select background color for log track by selecting a color from Color Picker dialog.

§  Zone Set: select a Zone Set to display zones boundaries on log track

§  Marker Set: select a marker set to display markers on the log track. Users can choose different marker set on tracks

11.2.2.1.4.                 Edit Curve properties

      I2G has 2 places to display properties of the curve:

·      Properties window: focus on a curve, you will see its properties in the Properties Editor window

Figure 11‑23 Curve Properties in Properties Editor window

·      Properties dialog: double-click or right-click on a curve and select curve properties option to open it.

Figure 11‑24 Open Curve Properties dialog

§  Name: allows the user to define the name of a curve to display into the log plot by manually typing text. Note: this name is not linked to the curve name in the data tree.

§  Show header checkbox: allows the user to tick on/off to display the header of the curve

§  Show dataset: tick on to display the dataset that contains the selected curve. The name will display as format @datasetName.@curveName

§  Left/Right value: allows the user to define the left/right value of a curve on a log track display. The family database defines the default scales of a curve.

§  Scale: user can choose to display curve as linear or logarithmic

§  Unit: user can choose the compatible unit to display

§  Family Scale: returns the Left/Right value to the family’s scale in database

§  Variable scale: returns the Left/Right value to the curve’s scale itself

§  Display As: choose the style of the display from the drop-down box:

+      Normal: display as normal type.

+      Cumulative: is used, for example, to plot lithology – type shading. The value plotted for a cumulative style curve is equal to the curve value plus the values of all other “cumulative” style curves in the log track whose names appear higher in the curve list than the curve is plotted.

+      Mirror: will plot the selected curve with its mirror image (same curve with reversed scale). The gap between the two curves is shaded with the selected curve color.

+      Pid: the PID style is designed to plot horizontal lines from the data point to the value zero

§  Block Position: allows displaying curve line with block type like a fluid curve

§  Wrap mode: If the values of the variable are outside of the horizontal axis limits, they are not displayed unless you activate this Wrap display mode. There are 4 options: None, Left, Right and Both.

§  Display mode: user can choose the type to display of curve as Line, Symbol, Both or None

§  Edit button: allows users to modify the color, width and line/symbol styles

11.2.7.1.    Create and edit a Marker Set in Log-plot

       In case you do not have the correct marker data and have to rely on the shape of the curves to define each marker by geological components, I2G helps you create marker data directly on the log-plot.

To work with the marker in the log-plot, you must have an existing log-plot first.

11.2.7.1.1.                 Display marker set on log-plot

Figure 11‑55 Select markers set in the Properties Editor window

11.2.7.1.2.                 Create markers on log-plot

Step 1.         Refer to this article to manually create new Marker Set in Marker Set Manager form

Step 2.         Create new or open an available log-plot from the data tree.

Step 3.         Show the newly created marker set to the log-plot by selecting it in the Properties Editor window.

Step 4.         You can start creating markers by right-clicking on log track > select Add Marker > the Choose Marker Template pops up.

 

Figure 11‑56 Choose Marker Template to add a marker on the log track

Tip: You can also open the Choose Marker Template by holding down Shift+Alt while left-clicking on log track

Step 5.         After selecting the marker template to be used > OK, move to the location you want to create a marker on log track, hold Shift key + left click to pick marker. To be able to capture the correct depth, you should enable the tooltip to work easier.

Step 6.         Hold down Shift+Alt while left-clicking on log track to switch to another Marker Template by choosing it from the Choose Marker Template dialog. Repeat step 5 to create new markers

Note: I2G will automatically save the changes of the marker set you are working on to the log-plot. So, you will not need to manually save the marker set.

11.4.     Log-Plot Template

11.4.1.         Custom Template

After creating a specific log-plot, to save time and homogenize data displayed, users can save it as a template and apply it to other wells/datasets to create a cross-section. The log-plot template saves all the settings of log-plot, the curves present in the tracks, the curves properties, and the track properties.

To save a log-plot as a template: in Log-plot window, click on Save As New Template () icon and enter a name for the new template. All of the templates will be stored in the Asset Manager.

Figure 11‑74 Save a log-plot as a template

To Apply a template for any other wells/datasets:

Step 1.         From Menu Bar, click on the Plot tab and choose Custom Template, the Select Log-Plot Template dialog opens.

 

Figure 11‑75 Select log-plot template dialog

Tips: From Data-tree, Users can also open the Custom Template dialog by right-clicking on Log-plot > New Log-Plot > Custom Template

Step 2.         In Log plot Template list box, select a template you want to apply for wells/datasets

Figure 11‑76 Choose a log-plot template from the dropdown box

Tip: Users can type 1 or more characters to search the template name quickly

Step 3.         Enter a name for the new applied log-plot. The name will be automatically suffixed with @wellName/@datasetName

Step 4.         Select an Insertion Level you want to apply the template. There are 2 levels:

·      Well: curves will be collected by exact syntax @datasetName /@curveName in those assigned wells

·      Dataset: curves will be collected by @curveName in those assigned datasets

Notice that there will be no display if curves do not satisfy the above conditions.

Figure 11‑77 Insertion level when applying template

Tips: Users can select multi-items by holding Shift/Ctrl key, all the items you specifically select will be highlighted.

Step 5.         Click Ok to launch the procedure. All the new log-plots now appear.

 

Figure 11‑78  Apply log-plot template to other wells/datasets

Note: The Ok button is only activated after you define all the options
