6.1.        Organize Zone Set

·      A zone set is a specific dataset storing your zones of interest as intervals/a collection of one or many zones at different intervals.

·      Each zone is defined by its information (top and bottom depth) and its template.

·      A zone set template specifies the display properties of a zone set.

·      A Zone Set Template is a collection of one or many zone templates. Each zone template contains the zone’s information such as name, background, foreground, and pattern.

·      Zone Set  ó Zone Set Template, Zone ó Zone Template.

·      Zone Set and Zone Set Template are jointly managed by the Zone Set manager form.

      To open Zone Set manager form: from Menu Bar, click on the Data tab and choose Zone Set Manager. The Zone Set Manager form will be displayed.

Figure 6‑1 Open Zone Set Manager window

6.1.1.   Zone Template Set

      Template Set tab allows users to format the display properties of the Zone Set. It includes fields like Background, Foreground, Pattern, and Name of the Zone Template.

      All Zone Template Sets are managed at the Template Set tab of the Zone Set Manager window. When modifying any property, the software will automatically update accordingly.

Step 1.         From the Template Set tab, click Create New Zone Template Set () icon on the top left corner. The New Zone Template Set dialog pops up.

Figure 6‑2 Create New Zone Template Set

Step 1.         Type your Zone Template Set name > click OK. The new Zone Template Set will be displayed on the left list of the window.

6.1.3.   Zone Set

      Zone Set tab manages all zone set data that you have in every single well of the project.

      All Zone Set properties are directly assigned from the zone set template. To view a zone set form on the right-hand side, click on the well and then select the desired zone set to display.

Figure 6‑10 Zone Set tab

6.1.3.1.        Create a new Zone Set

Step 1.         From the Zone Set tab, focus on a well you want to create a Zone Set for

Step 2.         Click Create New Zone Set () icon on the top left corner. The New Zone Set dialog pops up.

Step 3.         In the New Zone Set dialog, from the dropdown list, choose a Template Set that was created in Template Set tab (check out this article)

Figure 6‑11 Add a new Zone Set for a Well

Note: When select Auto Create Zones, the software will apply the selected template to create the corresponding zones and data for each zone. The intervals will be divided equally by the zone number defined in the Zone Template

Step 4.         Click OK. The new Zone Set is now added below the focused well in the data tree
