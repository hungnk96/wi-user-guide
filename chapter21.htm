Module 21:    Reservoir Analysis

21.1.     Hydraulic Flow Unit (HFU)

The module takes Core Porosity and Core Permeability data and divides it up into Hydraulic flow units based on the following methods Rock Quality Index (RQI).

Flow unit will be classified mainly based on Flow Unit Indicator (FZI) histogram. The FZI boundaries can be adjusted interactively. Once they have been picked an output HFU curve is created.

21.1.1.          Open HFU module

Step 1.         From the Menu Bar, select the Reservoir Analysis tab and choose the Hydraulic Flow Unit button. A dialog Open Hydraulic Flow Unit Analysis will be opened.

Step 2.         Users can open an existing model or create a new model by clicking the button () and type the model name. After creating a new model > click OK, an HFU window will be displayed.

21.3.     Saturation Height Modeling

A complete workflow for saturation Height Modeling (SHM) offered on i2G contains three steps: Data correction, Build Pc model and Apply Pc model. The user can create Saturation versus height functions from capillary pressure (Pc) data. The Saturation Modeling modules have a multi-well interface and enable you to review and QC data from multiple wells in a field/study area.  The module also allows you to QC the raw Pc measurements and makes corrections for overburden stress, clay-bound water and for the difference between laboratory and reservoir conditions. The Saturation Modeling module is comprised of 3 interrelated tools:

·      Data Correction: these tasks enable the user to perform data correction which includes core Pc and core water saturation. The output of these tasks is subsequently used in the Build Pc model module.

·      Build Pc Model: the task takes corrected/converted Pc and water saturation as inputs to build saturation height functions. Various utilities such as build function by rock types, manually grouped core samples, etc. There are three function types available in the R2 version including J Function, J Modified Function, and Corey Brooks.

·      Apply Pc Model: the task is used to generate Water saturation (Sw) versus Height functions from log curve data. This will take Free Water Level (FWL), fluid properties such as density, and porosity, rock type and permeability depending on methods used.

All tasks in the Geomechanics section follow a Task frame including inputs, parameters, output, and Input Selection tab.
