Module 9:       Asset Manager

Asset Manager manages all Assets of the project, including Task’s Parameter Set, Workflow Template, Log-plot Template, User Defined Lines, Depth Shift table, Baseline Shift table, Split Curve table, Palette, Plot models and Zone Set backup from Workflow.

All the Assets are stored in the hierarchical list along the left side (Tree view). To view an Asset, click on the well and then select the desired zone set to display

To open the Asset Manager window: From Menu Bar, click on the Data tab and choose Asset Manager, the Asset Manager window will be displayed.

Figure 9‑2 Components of the Asset Manager

9.1.        Organize Asset

9.1.1.   Modify an Asset

Step 1.         Expand items of the left tree view, click on an Asset you want to view. Accordingly, you can see its content in the Display Area.

Figure 9‑3 Open an Asset

Step 2.         Modify any parameters from the Display Area, then click Save ( ) icon on the top left corner to save changes.

Tip: Hover the mouse cursor over the lower right corner of the active cell selection until the cursor changes to a small black plus cross. Left-click and drag down to copy and paste the contents to the cells below

Note: A non-editable Asset that can be handled by exporting it to your desktop and using third-party apps.

9.1.2.   Asset Filtering

Click on Filter drop-down box and select one Asset to filter

Figure 9‑4 Asset Filtering

9.1.3.   Clone an Asset

Step 1.         From the Tree view, right-click on an Asset and choose Clone to new parameter option from its context menu.

Figure 9‑5 Clone an Asset

Step 2.         In the Clone to new parameter pop-up dialog, enter a name for the cloned Asset and then click Ok.

9.1.4.   Rename an Asset

Step 1.         From the Tree view, focus on an Asset

Step 2.         You can rename an Asset by clicking on Rename () icon in the top right corner or rename it from the Properties Editor window.

9.1.5.   Delete Assets

Step 1.         From the Tree view, right-click on one or more Assets and choose Delete option from its context menu

Figure 9‑6 Delete Assets

Step 2.         Click the OK button from the confirmation dialog.

9.2.        Palette configuration

Palette is a range of colors. I2G allows users to define palette as User Palette at the Asset Manager to color a curve display in plots.

Step 1.         In the Asset Manager window, click on New Palette (+) icon on the top left. The New Asset dialog pops up.

Figure 9‑7 Open Add New Palette dialog

Step 2.         Enter the name “Palette_Demo” for the new Palette and click OK. You can see the “Palette_Demo” item is added below the Palette in the Data tree.

Step 3.         Focus on “Palette_Demo” (1), add the first color for this one by clicking on () icon (2) in the top right corner. A new color will be listed in the table

Step 4.         Define a color by entering Red, Blue, Green, Alpha value (3) or click on the Black box to select a color from the Color Picker dialog (4)

Figure 9‑8 User-defined Palette

Step 5.         Continue to add more colors for your Palette following these above steps.
