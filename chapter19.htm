Module 19:  Conventional – Computation

In this module, you will compute the volume of shale (VSh) using two tasks: Shale Volume Gamma Ray and Shale Volume Neutron – Density. You will also learn how to compute porosity and saturation for multiple zones.

Learning Objectives

After completing this module, you will know how to:

o   Compute shale volume

o   Compute effective porosity

o   Compute saturation.

19.2.3.         Pickett Plot

The purpose of the Pickett plot is to help the user to determine Rw using a double logarithmic plot of a true resistivity measurement (on the X axis) versus a porosity measurement (on the Y axis).

In addition to the points that have appeared after the selection of the well(s), four lines are also present in the Cross-plot (see below). These lines correspond to the water saturations for different resistivities. The four lines correspond to water saturation from bottom to top of 100 %, 70 %, 50 % and 30 %. The slope of these lines is - m and the intersection of the 100 % Sw line with the porosity axis is a * Rw.

Equations:

·      Sw=1.00:log10(y)=(-1./m)*(log10(x)-log10(a*Rw))

·      Sw=0.2:log10(y)=(-1./m*)(log10(x)-log10(1*Rw)-log10(1/pow(0.2,2)))

·      Sw=0.3:log10(y)=(-1./m)*(log10(x)-log10(1*Rw)-log10(1/pow(0.3,2)))

·      Sw=0.5:log10(y)=(-1./m)*(log10(x)-log10(1*Rw)-log10(1/pow(0.5,2)))

To open Pickett Plot from Sw Task:

Step 1.         From the Sw Task interface, click Input Selection tab

Step 2.         Open a cross-plot between Porosity and Resistivity directly by clicking Crossplot button on the top sidebar

Figure 19‑24 Open cross-plot directly from Sw task

Step 3.         From the cross-plot interface, select Pickett plot tab then setup a, m, n parameters. When these parameters are adjusted, the positions of the lines will change until a satisfactory result is reached.

Figure 19‑25 Pickett plot

Tip:

-       User can use the log-plot to identify the water zones then confirm them on the Pickett plot

-       Pickett plot can be used as an independent tool to assess a, m, n, and Rw from a Resistivity–Porosity cross-plot.

19.3.     Summation

Based on user-defined cut-offs, the Summation algorithm determines the reservoir intervals, quality, and thickness.

19.3.1.         Summation Interface

From the Menu Bar, select Petrophysics tab > in the Clastic menu, select Summaries. The Summaries will be displayed.

 

Figure 19‑26 Open Summaries task from the Menu Bar

There are three dedicated mandatory inputs: Shale Volume, Porosity, and Water Saturation.

Note: these inputs are used as cut-off values.

      Reference for Summation

The Summation algorithm can be simultaneously launched using several references (MD, TVD, TVDSS) if available in the selected dataset. By default, the algorithm uses the MD as the reference of the selected dataset.

Output Configure tab allows cut-off curve selection for defining the output flags.

By default, ROCK, RES, and PAY are associated with the Shale Volume, Porosity, and Water Saturation curves

·      ROCK: Defines the shale-free depth intervals. By default, only the shale volume cut-off is active.

·      RES: Defines the potential reservoir intervals in the ROCK’s intervals. By default, the shale volume and the porosity cut-offs are active.

·      PAY: Defines the hydrocarbon impregnated reservoir intervals. By default, the shale volume, porosity, and water saturation are taken into account.

Figure 19‑27 Output Configuration tab in the Summaries task

Input Selection tab, the user can drag and drop multiple wells which are defined as a summary. Users can summarize by zone or by well

Figure 19‑28 Select the proper input curves in the Summaries task

In the Parameters sub-tab, you establish the cut-offs for each curve. By default, each curve has a predefined range. In this case, Shale Volume, Porosity, and Water Saturation curves have a predefined range between 0 and 1.

      If additional curves/families are used, the limits of their display range are extracted from their family databases and can be manually modified.

      The bounds can be modified in the predefined drop-down list. By default, the range is between the minimum and maximum cut-off values including the actual minimum value:

      Minimum cut-off <= Range of values < Maximum cut-off

      And is written as: min <= *value* < max in the Parameters tab.

Figure 19‑29 Set parameters for Summaries task

19.3.2.         Summation Tab

       Each of the Summation tabs (Summation MD, Summation TVD, Summation TVDSS) summarizes these measurements:

Attribute

Description

Gross

The thickness of the zone.

Net

The thickness of the interval defined by the cut-off values and the different flags that were used in the analysis.

Net To Gross

The ratio of net-to-gross.

BVW

Bulk volume of water. Equation: BVW = net average porosity * average water saturation.

POR-TH

Porosity thickness. Equation: POR-TH = net average porosity.

HCPOR-TH

Hydrocarbon * Porosity thickness for each selected zone.

Equation: HCPOR-TH = net * average porosity * average Hydrocarbon saturation.

Av Shale Volume

Averaged Shale Volume.

Av Porosity

Averaged Porosity.

Av Water Saturation

Averaged Water Saturation.
