Module 27:  Dashboard

27.1.     What is Dashboard in i2G

Dashboard is a data visualization tool that allows all users to understand the analytics that matter to their project. Data is visualized on a dashboard as bar charts, Pie and Doughnut so that users can visually tracks, analyzes and displays metrics and key data points to monitor the health of their business against benchmarks and goals.

27.2.     How to open a Dashboard

27.3.1.         Chart types

I2G provides 4 types of chart:

·      Bar chart (vertical): is a chart with rectangular bars with lengths proportional to the values that they represent. One axis of the chart shows the specific categories being compared, and the other axis represents a discrete value. Bar charts provide a visual presentation of categorical data which is a grouping of data into discrete groups. Use a bar chart to show categories being compared or to make generalizations about the data quickly

Figure 27‑2 Bar chart (vertical) in Dashboard

·      Horizontal Bar chart: is the same as a vertical bar chart, only the x-axis and y-axis are switched

Figure 27‑3 Horizontal Bar chart in Dashboard

·      Pie chart: is a circular statistical graphic, which is divided into slices to illustrate numerical proportion. Pie charts can be effective in showing the contributions of data segments as a percentage of a whole

Figure 27‑4 Pie chart in Dashboard

·      Doughnut chart: is a variation on a Pie chart except it has a round hole in the center which makes it look like a donut, hence the name.  Doughnut charts are meant to express a "part-to-whole" relationship, where all pieces together represent 100%

Figure 27‑5 Doughnut chart in Dashboard

27.3.2.         Data Sources

A Dashboard connects to your project data as 5 levels:

·      Well by Type

·      Well by Field

·      Well by Operator

·      Well by Tag

·      Curve by Tag

27.3.3.         Ticks

I2G allows users to set up a number of gridlines on the chart, the default value is 10

27.3.4.         Title

The title of the chart allows users to change the name by manually typing text.
