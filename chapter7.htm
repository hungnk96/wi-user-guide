7.1.        Organize Marker Set

·      Markers are used to mark the geological changes of rock along with the depth of an underground well.

·      A marker set is a collection of multiple markers.

·      The structure of each marker consists of marker depth and marker template.

·      The marker template specifies the display part of the marker including Name, Color, Line Width, and Line Style.

7.1.1.   Marker Template Set

       The Template Set tab is the determinant of the displaying properties of the Marker as well as the Marker Set. A Marker Template contains information fields including Name, Color, Line Width, and Line Style. Template settings will be managed in the Template Set tab of the Marker Set Manager window.

      From the Template Set tab, you can change every displaying property of each Marker. When modifying any property of the Marker Template, the software will automatically update the Marker(s) using that Marker Template.

7.1.1.1.        Create new Marker Template Set

Step 1.         From the Template Set tab, click Create New Marker Template Set () icon on the top left corner. The New Marker Template Set dialog pops up.

Figure 7‑1 Create a New Marker Template Set

Step 1.         Type your Marker Template Set name > click OK. The new Marker Template Set will be displayed on the left list of the window.

7.1.3.   Marker Set

      Marker Set tab manages all Marker data that you have in every single well of the project.

      The Marker name, Marker Set name and other Marker’s properties are assigned by the corresponding Marker template. To view a Marker Set on the right-hand-side, click on the well and then select the desired Marker Set to display.

Figure 7‑9 Marker Set tab

7.1.3.1.        Create new Marker Set

Step 1.         From the Marker Set tab, focus on a well you want to create a Marker Set for

Step 2.         Click Create New Marker Set () icon on the top left corner. The New Marker Set dialog pops up.

Step 3.         In the New Marker Set dialog, from the dropdown list, choose a Template Set that was created in Template Set tab (check out this article)

Figure 7‑10 Add a new Marker Set for a Well

Note: When select Auto Create Markers, the software will apply the selected template to create the corresponding Markers and data for each Marker. The intervals will be divided equally by the Marker number defined in the Marker Template

Step 4.         Click OK. The new Marker Set is now added below the focused well in the data tree
