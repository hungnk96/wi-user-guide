Module 2:       Getting Started

This chapter explains how to start using i2G software and describes the I2G basic functionality.

Learning Objectives:

·      Managing your account

·      A quick-start guide to the I2G user interface

·      A brief description of the data structure

2.1.        Managing your account

2.1.1.   Log into i2G account

Step 1.         Go to https://wi.i2g.cloud/

Step 2.         At the Sign In the dialog, enter your Username and Password.

Figure 2‑1 Sign In dialog

Tip: Select Remember me option then i2G will remember you for a long time and the website won’t ask you for a password again.

Step 3.         Click Sign In.

If you’re having trouble logging in, please contact your administrator or contact us via i2g.cloud/contact-us

2.1.2.   Log out of i2G

Step 1.         Click Log Out ( ) icon at the top right of i2G page

Step 2.         Click Ok at the Log Out dialog that appears

Figure 2‑2 Log Out dialog

2.1.3.   Change your password

To change your password on i2G if you're already logged in:

Step 1.         Click on your account name in the top right corner (the left side of Log Out button), Change Password dialog pops up

Step 2.         Enter your Old Password, New Password and Confirm Password.

Figure 2‑3 Change Password dialog

Note:

-       The New Password must contain at least 8 characters

-       The New Password and the Confirm Password must be the same

Step 3.         Click Ok.

2.2.        I2G User Interface

Figure 2‑4 Primary components of i2G

I2G consists of the following primary components which are described in the section below:

·      Menu Bar for accessing domain menus

·      The Toolbar allows you to quickly access tools under Menu Bar

·      Pop-out windows:

§  Project Browser– Listing all project data in the hierarchical Data tree

§  Properties Editor window – Listing all properties of the selected object

§  Output window: Displays messages from the system

·      Display area: Area where all objects display such as log-plot, cross-plot, histogram, workflow, task…

2.2.1.   Menu Bar

The i2G Menu Bar is located at the top of the window. Including:

·      Home

·      Data

·      Plot

·      Data Processing

·      Petrophysics

·      Geomechanics

·      Reservoir Analysis

·      Machine Learning

·      Help

Click a menu name with the left mouse button to display the Tool Bar.

The icons at the right end of Menu Bar allow you to manage your i2G’s account

Tip: click ( ) to hide the Tool Bar

2.2.2.   Tool Bar

These icons allow you to quickly access i2G tools under the Menu Bar. You can launch modules such as Pre-Computation, Clastic, Fracture Basement, Geomechanics, FPress, SHM, ML, and others.

Figure 2‑5 Tool Bar in i2G

2.2.3.    Dock window

2.2.3.1.        Project Browser

      From Menu Bar, click the Home tab and choose the Project Browser. The Project Browser dock window will be displayed on the left side of the i2G window.

      The Project Browser functionality allows the user to interpret an i2G project in a graphical tree view interface. The view is hierarchical, similar to a typical Windows browser.

The Project Browser displays all the data in the project in a set of hierarchical folders. Similar or related data is grouped together in one of several main folders. These main folders can contain subfolders where the details for each type of data are stored.

There are 5 areas: Wells, Log-Plot, Cross-Plot, Histogram, and Workflow.

Figure 2‑6 Project Browser window

The Project Browse allows the user to apply numerous actions such as data importing, data filtering, tagging at:

·      The Well level

·      The Dataset level

·      The Zone Set level

·      The Marker Set level

·      The Image Set level

·      The Variable level

The available actions in the context menu depend on the object you select

 

Figure 2‑7 Context menu of the objects on data tree

2.2.3.2.        Properties Editor

      From Menu Bar, click the Home tab and choose the Project Browser. The Properties Editor window will be displayed right under the Project Browser.

Properties Editor window displays all parameters of the selected object. Its structure may vary depending on the object being selected at that moment. Most i2G objects can be retrieved, modified, and configured from this window, including the units of a variable and parameters.

Figure 2‑8 Properties Editor window

In addition, users can also modify properties of an object from Properties dialog by double-clicking on the desired object. In this dialog, every option and setting will be displayed including advanced, complicated ones. With this, every object in the log view is fully customizable

Figure 2‑9 Track Properties dialog

Figure 2‑10 Curve properties dialog

2.2.3.3.        Output

      From Menu Bar, click the Home tab and choose Output. The Output window will be displayed right under the Properties Editor window.

The output window displays messages from the system including information, errors, and warnings. It allows you to diagnose issues when working in i2G

2.3.        Organize workspace

The windows in i2G can be moved around at your convenience. If more than one window is put in the same location, the windows are placed on top of each other and tabs will be shown.

If data extends beyond the visible area, you can move it into view using the scroll bars. The window remains open until you close the window. The dashed double line, called a “handle”, at the top of the window indicates that the window can be “torn off”, opening a separate window that can be placed somewhere else on the screen.
