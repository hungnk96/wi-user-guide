Module 16:  Task

      In i2G, a task is referred to as a package of a calculation procedure to convert input curve(s) into desired G&G information. The user can create more than one Task and then link them together in a WorkFlow for a total customized by the user.

Figure 16‑1 Petrophysic tasks

Learning Objectives

   After completing this Best Practice, the user will know how to:

o   Manually create a Task.

o   Input curve into a Task.

o   Run the calculation.

16.2.     Parameter management

After interpreting a specific task, to save time, users can save it as a parameter and apply it to other workflows/wells/zones.

Figure 16‑18 Parameter manager dialog

I2G distinguishes between several types of parameters.

·      Flow parameters: a parameter is used as a workflow parameter.

·      Project parameters: an identical value for the entire project, such as all wells in all zones.

·      Well parameters: a unique value for the well throughout all ones.

·      Zone parameters: an identical value to the zone throughout all well.

16.2.1.         Save parameters

Step 1.         From the Task, click Input Selection tab > click Parameter sub-tab

Step 2.         In the table, left-click and drag down to select the desired parameters you want to save > right click and choose Define as zone parameter option from its context menu. The Parameter Manager dialog pops up.

Step 3.         Click Save () from the top navigation bar. The Save Parameter Set as New dialog pops up

Figure 16‑19 Save Parameter Set as New dialog

Step 4.         Enter a name for the parameter set, click Ok.

Note: All of the Parameter Sets will be stored in the Asset Manager.

16.2.2.         Load parameters

Parameter files are loaded into a workflow with two steps: (1) Load the file into the Parameters manager and (2) Propagate parameters in the workflow.

Step 1.         From the Task, click Input Selection tab > click Parameter sub-tab

Step 2.         In the table, right click and choose Parameter Manager option from the context menu. The Parameter Manager dialog pops up.

Figure 16‑20 open Parameter Manager dialog from Task

Step 3.         Click Load ( ) (1), the Select Parameter Set pops up

Step 4.         Choose a Task method > choose a desired Parameter Set > choose Load level (Project, Zone, Well or Flow) you want to load data from (2).

Step 5.         Click  (3). All parameters from the selected parameter set will appear in the Parameter Manager dialog.

Figure 16‑21 Select Parameter Set to be loaded

Step 6.         After all the parameters have been loaded into the Parameters manager, click Propagate parameters (4) from the level you want such as Flow (), Project ( ), Wells (), Zones ( ) to the current Task.

Figure 16‑22 Propagate parameters

Note: You can load a saved parameter set with the correct parameters only if the parameter set and the destination task have the same parameter name
